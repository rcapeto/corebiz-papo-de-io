## 🚀 How to execute

Clone the project and access the folder.

```bash
$ git clone https://github.com/rcapeto/corebiz-papo-io.git
$ cd corebiz-papo-io
```

To start, follow the instructions:
```bash
# Link your project
$ vtex link
```
The app will be available in the [web address](http://localhost:3333).
